# Frontend Design

Frontend Design is a Finch mini tool that bundles the `frontend-design` skill and provides a Composer shortcut for starting polished web UI work.

## Usage

Enable the mini tool, then ask Finch to build or improve a website, page, component, dashboard, or other frontend interface. You can also choose **Frontend Design** in the Composer toolbar to insert a design brief.

## Included skill

- `frontend-design` — creates distinctive, production-grade frontend interfaces with a clear visual direction.
