# 网页设计

网页设计是一个 Finch 小工具，内嵌 `frontend-design` 技能，并提供用于发起精致网页 UI 设计任务的 Composer 快捷入口。

## 使用

启用小工具后，直接让 Finch 创建或优化网站、页面、组件、仪表盘等前端界面；也可在 Composer 工具栏选择「网页设计」插入设计请求。

## 内嵌技能

- `frontend-design`：以清晰的视觉方向创建具有鲜明风格、可用于生产的前端界面。

## 来源与许可证

内嵌的 `skills/frontend-design/` 内容未作修改，使用 [Apache License 2.0](skills/frontend-design/LICENSE.txt)。本小工具新增的 Finch manifest、多语言文案和 Composer 操作同样使用 Apache-2.0。

## 开发

```bash
npm install
npm run build
npx @finchtoys/minitools doctor .
```
