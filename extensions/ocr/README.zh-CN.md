# PP-OCRv6

基于 Python PaddleOCR 的高精度文字识别扩展，支持图片和扫描版 PDF。

## 使用方式

1. 在 Finch 中启用 **PP-OCRv6** 扩展。
2. 发送图片或 PDF，说：
   - `识别这张图里的文字`
   - `提取这个 PDF 的文字`

首次使用自动安装依赖（Python venv + pip install），无需手动操作。

## 工具

| 工具 | 说明 |
|------|------|
| `ocr_image` | 从图片（PNG、JPG、WebP、BMP 等）提取文字 |
| `ocr_pdf` | 逐页 OCR 扫描版 PDF |
| `setup_ocr` | 检查环境并安装依赖 |
| `ocr_status` | 快速健康检查 |

## 环境要求

- **Python 3.10 – 3.12**
- 依赖（`paddleocr`、`paddlepaddle`、`PyMuPDF`、`opencv-python`、`numpy`）首次使用时自动安装

## 性能与上限

| 类型 | 建议上限 | 说明 |
|------|---------|------|
| 单张图片 | 原始尺寸无限制 | 最长边 >3000px 自动缩放，几乎不占额外内存 |
| PDF 页数 | 数百页 | 200 DPI 渲染，>98% 空白页自动跳过 |
| 单页 PDF 尺寸 | A0 以下 | 超大图纸建议拆分为单页处理 |

- **多页 PDF**（>8 页）自动启用 4 线程并行处理
- PDF 输出带页号标题和置信度
- 模型首次加载较慢（~10 秒），后续调用快
